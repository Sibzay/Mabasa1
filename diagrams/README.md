## How to View Diagrams

### Option 1: VS Code with Mermaid Plugin
1. Install the "Mermaid Preview" extension in VS Code
2. Open any `.md` file in this directory
3. Use Ctrl+Shift+P â†’ "Mermaid: Preview" to view the diagram

### Option 2: Online Mermaid Editor
1. Copy the Mermaid code from any `.md` file
2. Paste into https://mermaid.live/
3. The diagram will render automatically

### Option 3: GitHub/GitLab
- GitHub and GitLab automatically render Mermaid diagrams in Markdown files
- Just open the file in your repository

## Screenshot Instructions

1. Open the desired diagram file
2. Use your preferred method to render the diagram
3. Take a screenshot of the rendered diagram
4. Each diagram is in its own file for easy capture

## Diagram Overview

- **Flow Diagrams (01-07)**: Show data flow through different user interactions
- **ER Diagram (08)**: Shows database relationships and schema structure
- **Class Diagram (09)**: Shows backend Django models and frontend Flutter classes with relationships
- **Use Case Diagram (10)**: Shows all user interactions and system functionalities for both Employee and Employer roles
- **Interface Design Diagram (11)**: Shows UI structure, navigation flow, and screen relationships with detailed interface specifications

All diagrams use Mermaid syntax and are ready to render in any Mermaid-compatible viewer.
